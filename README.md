# STEM Booking System

## Overview
The STEM Booking System is a web application designed to facilitate the booking and management of STEM-related resources and activities. It serves as a platform for users to create, view, and manage bookings for STEM events, labs, and equipment.

## Features
- **User Registration and Login:** Users can create accounts and log in to manage their bookings.
- **Resource Management:** Administrators can add and manage STEM resources available for booking.
- **Booking System:** Users can book resources for specific time slots, view available resources, and cancel bookings if necessary.
- **Notifications:** Users receive notifications for upcoming bookings and any changes to their reservations.
- **Admin Dashboard:** A comprehensive dashboard for administrators to oversee bookings, resources, and user management.

## Technologies Used
- Frontend: HTML, CSS, JavaScript
- Backend: Node.js, Express
- Database: MongoDB

## Installation
1. Clone the repository: `git clone https://github.com/NataliaK36/stem-booking-system.git`
2. Navigate to the project directory: `cd stem-booking-system`
3. Install dependencies: `npm install`
4. Start the application: `npm start`

## Usage
After starting the application, navigate to `http://localhost:3000` in your web browser to access the STEM Booking System.
